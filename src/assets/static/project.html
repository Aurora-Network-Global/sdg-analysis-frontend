:: SDG Query Executor

Bibliometrics of Relevance

In short:

The query executor helps the project to automate the collection of data. it is a tool written in python to collect data from Scopus, adding data from Unpaywall and Altmetric.

Impressum

The Query Executor is used as an internal tool by the Aurora SDG Analysis team.

The Software is developed by Eike Spielberg from the University Library of Duisburg-Essen. The source code for the frontend and backend are available via github.

The Query Executor is hosted by the Vrije Universiteit Amsterdam .

Contact

In case of problems or bugs please contact Eike Spielberg, University Library of Duisburg-Essen or Maurice Vanderfeesten, Vrije Universiteit Amsterdam
